# bidirectional-scarcity-with-onchain-seeds-and-tiers

By merging SatoshiArt’s tiered pigment pricing with GenP5’s p5.js‑based, Lightning‑native generator, artists can make both the pigment and the generative algorithm scarce. Each pixel not only costs according to its rarity tier but also derives from an on‑chain seed that can be micro‑paid for each generation. This bidirectional scarcity produces unique “pixel artifacts” whose visual and.Arrayable lineage are both immutable and monetizable on Lightning.
