# multi-language-context-requires-consistency

When an assistant shifts from English to Spanish mid-response without explicit user request, it creates a language barrier that interrupts the user's workflow and can cause confusion. The friction occurs when the assistant's language choice doesn't match the user's communication pattern, as seen when responding to English-speaking users with Spanish text. Maintaining language consistency throughout interactions is essential for seamless user experience.
