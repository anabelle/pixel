# response-deviates-from-user-instructions

Multiple observations show the assistant providing extraneous or overly verbose answers (e.g., full debriefs, detailed data dumps) when the user requested concise output, or completely omitting requested content such as a single‑sentence observation. This pattern highlights a lack of strict adherence to user directives and a tendency to extend or alter the response beyond what the user asked for.
