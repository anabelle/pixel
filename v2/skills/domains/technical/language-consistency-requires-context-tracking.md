# language-consistency-requires-context-tracking

Multiple instances show the assistant abruptly switching from English to Spanish mid-conversation, creating barriers that disrupt the workflow. Maintaining a stable language context is critical for user trust, as inconsistent output confuses the user and breaks the flow of information exchange.
