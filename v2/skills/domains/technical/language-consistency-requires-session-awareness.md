# language-consistency-requires-session-awareness

Multiple instances show the assistant shifting to Spanish or responding in Spanish while the user communicates in English, creating a language barrier that confuses the user and disrupts the interaction flow. This indicates a failure to track and persist the user's language preference contextually. The system needs to enforce strict adherence to the user's established language to maintain seamless communication.
