# response-mismatch-with-single-sentence-request

Across multiple captured interactions, users explicitly requested a brief or single‑sentence observation, yet the assistant generated complete debriefs or full reports. This pattern shows a consistent failure to honor brevity requirements, leading to unnecessary verbosity and user frustration.
