# output-always-match-user-intent

Several interactions reveal that the model generates overly detailed reports or unrelated questions when the user explicitly states a preference for a concise executive debrief or a single command line. This mismatch suggests a systematic lack of intent detection or format enforcement when handling user directives, leading to frustration and stalled conversations.
