# error-context-neglect

When users reported recurring model errors or specified that the assistant should be aware of a model‑unsupported error, no acknowledgment or corrective reference appeared in the assistant’s responses. This omission can cause the assistant to fail to adjust its behavior (e.g., retry logic or alternative model invocation), leading to incomplete transactions or abrupt session exits.
