# error-context-acknowledgment-lacking

Multiple user reports of persistent `model_not_supported` errors were ignored or unaddressed by the assistant. This lack of acknowledgment prevents users from diagnosing or escalating issues properly and can hide additional underlying problems. Ensuring the assistant admits known failures would enhance user trust and facilitate faster resolution.
