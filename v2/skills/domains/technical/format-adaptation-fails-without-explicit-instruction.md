# format-adaptation-fails-without-explicit-instruction

When users request specific formats like "short trend report" or "executive debrief," the assistant provides generic responses instead of following the requested format. This occurs because the assistant doesn't recognize format requirements as explicit instructions requiring template adherence, leading to mismatched outputs that don't meet user expectations.
