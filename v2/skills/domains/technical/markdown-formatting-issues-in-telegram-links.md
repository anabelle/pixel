# markdown-formatting-issues-in-telegram-links

Users attempting to share links with markdown formatting in Telegram may encounter errors due to the platform's lack of markdown support. This can result in broken links or display issues when extra characters like quotes are included. Clear communication about platform limitations is necessary to prevent user frustration.
