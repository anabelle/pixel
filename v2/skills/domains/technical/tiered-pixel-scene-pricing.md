# tiered-pixel-scene-pricing

Using the SatoshiArt model, each pixel’s color is stored permanently on‑chain and sold at different satoshi tiered rates—from common (~1 k sats) to legendary (~25 k sats). This dynamic pricing makes the palette itself scarce, contrasting typical flat‑rate systems that treat pixels uniformly. The result is a self‑reinforcing value model where the rarity of choices drives economic activity on the canvas.
