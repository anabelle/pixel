# assistant-proactive-search-need

When users ask for new articles or research findings, the assistant often fails to provide options from the web, forcing the user to perform their own search. This friction wastes time and diminishes the assistant’s perceived usefulness. An improved design should automatically expand queries, recommend alternate sources, and solicit clarification if results are empty.
