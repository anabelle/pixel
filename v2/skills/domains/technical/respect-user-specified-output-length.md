# respect-user-specified-output-length

Multiple recorded interactions reveal users asking for a single sentence or brief executive debriefs, yet the assistant produced full debriefs or extended reports, creating a communication break. Enforcing output‑length constraints when the prompt includes explicit brevity instructions would align responses with user intent and reduce friction.
