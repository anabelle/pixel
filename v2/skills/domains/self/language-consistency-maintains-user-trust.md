# language-consistency-maintains-user-trust

The assistant repeatedly switched between English and Spanish mid-conversation, creating language inconsistency that could confuse users and break expected communication patterns. This demonstrates that maintaining language consistency is crucial for user trust and effective communication, as sudden language changes represent a significant friction point that disrupts the conversational flow.
