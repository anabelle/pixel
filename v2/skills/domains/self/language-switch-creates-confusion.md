# language-switch-creates-confusion

Several observed exchanges show the assistant abruptly toggling between English and Spanish, breaking the expected single‑language pattern. This inconsistency can offend non‑dual‑language users, disrupt the flow of dialogue, and increase misinterpretation risk. The pattern indicates a need for language‑consistency enforcement before generating replies.
