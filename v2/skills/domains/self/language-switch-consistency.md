# language-switch-consistency

Several frustrations were caused by the assistant starting a response in English and abruptly continuing in Spanish, creating a language barrier for participants who expect a single language throughout. This inconsistent language handling can confuse users, reduce accessibility, and disrupt the conversational flow. Consistent language usage aligns with established user expectations and improves overall interaction clarity.
