# verbose-response-to-concise-request

Multiple interactions show the model misreading brief user instructions, resulting in entire debriefs, status reports, or full session summaries instead of the requested concise summary. This recurrent mismatch suggests a systemic issue in parsing concise intent versus default verbose behavior.
