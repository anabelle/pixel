# ambiguous-task-switching-delay

In conversations where the user first requests a meta‑observation and then immediately asks for research findings, the assistant momentarily stalls to reconcile the two intents, creating a brief but noticeable lag. This interruption leads to user frustration and a perception that the assistant is “hesitant.” Clear prompt parsing mechanisms or a contextual state tracker could reduce such delays, improving flow.
