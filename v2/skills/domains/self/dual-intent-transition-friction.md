# dual-intent-transition-friction

Friction occurs when a user combines a request for a meta-observation about the interaction with an immediate demand for specific research data. This requires the assistant to resolve two different abstraction layers simultaneously, leading to a processing pause as it determines whether to prioritize the structural critique or the data retrieval.
