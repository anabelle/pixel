# markdown-support-requires-platform-awareness

When users encounter formatting errors with links containing extra quote characters, the assistant should immediately recognize that certain platforms like Telegram don't support markdown formatting. This requires understanding platform-specific capabilities to provide accurate troubleshooting rather than generic responses about link syntax.
